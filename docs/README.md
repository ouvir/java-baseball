# 구현 할 기능들 목록
1. 컴퓨터의 숫자 3자리를 만드는 기능(중복X, 순서 상관 O)
2. 사용자에게 숫자 3자리를 입력받는 기능 (예외처리 필요)
3. 사용자에게 받은 **문자**와 **정답**을 **비교**하는 기능
4. 3스트라이크시, 종료문구를 출력하는 기능
5. 다시하기 or 종료를 입력받는 기능
